## Quick start

Install it:

```bash
npm i tsup-preset-solid
# or
yarn add tsup-preset-solid
# or
pnpm add tsup-preset-solid
```

Use it:

```tsx
import tsup-preset-solid from 'tsup-preset-solid'
```
